# CSS-Basic
```
(#) p[title] = all p with title  attribute defined
(#) p[title='Welcome msg'] = all P with title='Welcome msg'

(#) p span{ // span inside of p

}

(#) h1 > p { //all p that are direct child of h1(direct child)

}

(#) p, h1{ // all p and h1

}

(#) p:hover{  // pseudo class(state of the element)

}

(#) p::first-line{ // pseudo element(part of an element)

}

```
